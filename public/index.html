Web Dice API

This web tool is designed to roll some dice.

URL:

/roll

Method:

GET

URL Params:

action=[string]

Where [string] is a text string indicating the action to be taken for the dice when rolled. If omitted, the application defaults to 'add'. Invalid actions return a 400 error. Options supported are:

  • add: Adds the reults of all dice.
  • count: Counts the number of dice that exceed a given value.

num=[number]

Where [number] is the number of dice to be rolled. Defaults to rolling a single die if omitted.

type=[number]

Where [number] is the number of sides on the dice to be rolled. Defaults to 10-sided dice if omitted.

plus=[number]

Used only when action=add. Where [number] is a number that will be added to the final sum. Defaults to 0 if omitted.

atleast=[number]

Used only when action=count. Increments the count by 1 for each die that rolls a value of at least [number]. If [number] is greater than the value of the 'type' parameter, or is omitted, then 'atleast' will be set equal to the 'type' parameter.

explodes=[number]

Indicates "exploding" dice, where dice are re-rolled if they roll a value of [number] or higher. When set to zero, no re-rolling takes place. If action=add, then both values are added to the total, and the default value is 0. If action=count, then either or both rolls may increment the count, and the default value is equal to the type parameter, so rolling the maximum results in a re-roll.

numexplodes=[number]

Where [number] is the maximum number of re-rolls that can be triggered by "exploding" dice. If numexplodes=0, there is no limit to the number of re-rolls. Defaults to 0 if omitted.

format=[string]

Where [string] is a text string indicating the format used when delivering the dice results. If omitted, the application defaults to 'html'. Invalid format options return a 400 error. Options supported are:

  • html: Returns results in human-readable HTML.
  • json: Returns JSON object with results described. The [object].result property contains the final results of the roll.
