Web Dice API

This web tool is designed to roll some dice.

URL:

/roll

Method:

GET

URL Params:

dice=[string]

Where [string] is in the format ##d##p##, with the first ## being an integer representing the number of dice, the second ## being an integer representing the number of sides on each die, and the third ## being an integer representing the number that should be added to the total.

If the 'd' is omitted, the application will defualt to rolling 10-sided dice. If the 'p' is ommitted, the application will default to adding zero to the total. If non-numeric characters other than a single 'd' and 'p' are included, the application will make a rudimentary effort to extract numbers, but results are formally undefined.

Examples

  • dice=1d10: Rolls a single 10-sided die.
  • dice=1d10p12: Rolls a single 10-sided die and adds 12.
  • dice=4p5: Rolls 4 10-sided dice.

format=[string]

Where [string] is a text string indicating the format used when delivering the dice results. If omitted, the application defaults to 'html'. Invalid format options return a 400 error. Options supported are:

  • html: Returns results in human-readable HTML.
  • json: Returns JSON object with results described. The [object].result property contains the final results of the roll.
